## 🛠️ How to Run

1. Clone the repository.
2. Open the desired program in your preferred IDE or editor.
3. Compile the program using a C compiler:

```bash
gcc filename.c -o program
```

4. Run the executable:

```bash
./program
```

(On Windows, run `program.exe`.)
